### 🚀 Cobalt Launcher Nano

Лаунчер для Minecraft на Python.
Устанавливайте ванильные версии, Forge, Fabric — всё в отдельные папки.

### ✨ Возможности

    ✅ Установка любых версий Minecraft (альфа, бета, снапшоты, релизы)
    🔧 Автоматическая установка Forge и Fabric как отдельных версий
    🧩 Поддержка модов – поиск и установка с Modrinth, избранное
    👤 Управление оффлайн-аккаунтами – добавление, переименование, удаление
    ⚙️ Настройка Java – автоматическая загрузка (8/11/17/21), ручной путь, аргументы JVM
    📁 Быстрый доступ к папкам – моды, ресурспаки, миры, скриншоты, конфиги
    💾 Резервное копирование миров, ресурспаков, конфигов и модов
    🔌 Плагины – расширяйте функциональность своими скриптами
    📝 Заметки – встроенный блокнот для важной информации
